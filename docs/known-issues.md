# 크리티컬 이슈 & 기술 부채

## 현재 남은 이슈

### 1. Gemini 배경 제거 품질 & 비용 한계
**문제**: `gemini-2.5-flash-image`는 이미지 생성 모델이라 배경제거가 재현성이 낮음
- 박스 위 상품 → 박스 일부가 상품으로 인식되기도 함
- 흰 배경 지시해도 살짝 회색빛
- 같은 이미지에 재실행하면 조금씩 다름
- **비용도 높음**: 이미지 출력 $0.039/장 → 배경제거 1회당 약 $0.04

**현재 대응**
- 프롬프트 강화 (객체 수/포즈/위치 유지 명시)
- 클라이언트 후처리 `whitenNearWhite()` — threshold 245↑ + 저채도 픽셀을 #FFFFFF로 치환

**근본 해결 후보 (우선순위 순)**
| 옵션 | 비용/장 | 품질 | 비고 |
|------|--------|------|------|
| **@imgly/background-removal** (클라이언트 WASM) | **무료** | 中~上 | 첫 로드 40MB, 모바일 느림 |
| **Recraft `recraft-remove-background`** | $0.01 | 上 | 투명 PNG, 후처리 불필요, Gemini의 1/4 가격 |
| BRIA RMBG 2.0 (Replicate) | ~$0.058 | 上 | 프리미엄, 오버스펙 |

데모 기간 후 Recraft 또는 @imgly로 전환 검토.

### 2. Vercel 함수 타임아웃
**문제**: `/api/render`에 `maxDuration = 60` 설정했지만, Hobby 플랜은 10초 제한

**영향**: 이미지 10장 + 폰트 로딩 시 타임아웃 가능

**현재 대응**
- 클라이언트에서 이미지 미리 압축 → 서버 처리 시간 단축
- 상하단 이미지는 서버에 안 보냄 → payload 감소

**근본 해결**
- Vercel Pro로 전환 (60초)
- 또는 Railway 같은 곳에 렌더 서버 분리

---

### 3. Google OAuth Preview 배포 로그인 불가
**문제**: Preview URL이 랜덤이라 OAuth redirect_uri_mismatch

**현재 대응**
- Preview 환경에만 `SKIP_AUTH=true`, `NEXT_PUBLIC_SKIP_AUTH=true` 설정
- Production에서는 정상 작동

**근본 해결 (필요 시)**
- Vercel Pro + 고정 Preview 도메인 할당

---

### 4. 모바일 미지원
**문제**: 3열 그리드 + 450px/480px 사이드 패널 고정 → 모바일 화면에서 사용 불가

**영향**: 데모 유저가 모바일 접속 시 UX 불가

**대응 (Phase 2)**
- 반응형 레이아웃으로 전환 (브레이크포인트 768px)
- 모바일은 탭 전환 방식 (사이드 패널 → 모달/드로어)

---

### 5. 크레딧 시스템 초기 사용자 대응
**문제**: 처음 접속한 유저가 크레딧 개념을 모름

**대응 (데모 중)**
- 프로필 패널 에 기능별 단가 표시 (상세 1, 이미지 5, 배경 5)
- 크레딧 부족 시 에러 메시지에 초기화일 안내 ("N월 1일 00:00 초기화")

**추가 개선 (선택)**
- 첫 접속 시 튜토리얼 모달
- "남은 크레딧" 헤더 상시 표시

## 기술 부채

### 코드 품질

| 항목 | 위치 | 해결 방향 |
|------|------|----------|
| `render.service.ts`와 `DetailPagePreview.tsx` 레이아웃 중복 | 서버/클라 양쪽에 동일 스펙 명시 | JSON 설정으로 추출해서 공유 |

| 배경제거 순차 처리 | `processAllImages`가 for 루프로 순차 호출 | `Promise.all` 병렬 처리 (Gemini rate limit 주의) |

## 비용 예상 (데모 기준)

### Gemini 단가 (2025-04 기준)
- 텍스트 입력/출력: $0.30 / $2.50 per 1M tokens
- **이미지 출력: $0.039/장** (≈ $30/1M tokens)

### 작업별 단위 원가
| 작업 | 비용 | 비고 |
|------|------|------|
| 상세페이지 통합 생성 | ~$0.007 | 이미지 5장 입력 + 텍스트 출력 |
| AI 모델 이미지 생성 | ~$0.04 | 이미지 출력 1장 |
| **배경 제거** | **~$0.04** | 이미지 출력 1장 (비중 큼) |

### 월 시나리오
| 시나리오 | 월 비용 (현재 Gemini) | Recraft 전환 시 |
|---------|---------------------|----------------|
| 100명 × 40% 사용률 | ~22만원 | ~10만원 |
| 150명 × 40% 사용률 | ~33만원 | ~15만원 |
| 150명 × 100% 사용률 (최악) | ~82만원 | ~38만원 |

**Google Cloud 예산 한도 월 50만원 설정 상태**.

> 배경제거를 Recraft로 전환하면 해당 기능 비용이 **$0.04 → $0.01** (4배 ↓). 데모 이후 가장 유력한 최적화 포인트.
